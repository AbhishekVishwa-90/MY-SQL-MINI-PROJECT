# MY-SQL-MINI-PROJECT
# 🎓 MySQL Student Record Management System

## 📌 Project Overview

The **MySQL Student Record Management System** is a beginner-friendly database project developed using **MySQL**. The main objective of this project is to understand and practice fundamental SQL operations by creating and managing a student database.

The project contains **100 student records** and demonstrates how SQL is used to create tables, insert records, modify table structures, update data, delete records, and retrieve information using different SQL queries.

This project is ideal for students, beginners, and anyone learning database management and SQL.

# 🚀 Project Features

* ✅ Created a MySQL database
* ✅ Created a student table with appropriate data types
* ✅ Inserted 100 student records
* ✅ Renamed the table from **STUDENT_DATA** to **STUDENT_RECORD**
* ✅ Added and removed the **Admission_Date** column

# 📂 Database Structure

## Table Name

`STUDENT_RECORD`

### Columns

| Column Name    | Data Type    | Description     |
| -------------- | ------------ | --------------- |
| Student_ID     | INT          | Primary Key     |
| Student_Name   | VARCHAR(100) | Student Name    |
| Age            | INT          | Student Age     |
| Gender         | VARCHAR(10)  | Male/Female     |
| Department     | VARCHAR(50)  | Department Name |
| Course         | VARCHAR(50)  | Course Name     |
| Year           | INT          | Academic Year   |
| Mobile_No      | VARCHAR(15)  | Mobile Number   |
| Email          | VARCHAR(100) | Email Address   |
| City           | VARCHAR(50)  | Student City    |
| Admission_Date | DATE         | Admission Date  |
